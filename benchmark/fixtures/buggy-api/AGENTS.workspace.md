## Team conventions

- Route files live in `src/routes/` — one file per resource
- The in-memory `db` object in `src/db.js` is the only data store
- Tests use `node:test` and `node:assert/strict` — no test framework dependencies
- All responses are JSON; use `res.json()` not `res.send()`
